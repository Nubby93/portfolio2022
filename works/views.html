Views

Mobile app design

ABOUT THIS PROJECT

Views is an art inventory web app set to give art dealers a quick and efficient way to manage their inventory.

The challenge

Managing the art inventory is difficult as you need to take note of each detail. It is a time consuming activity that needs to be done for each piece individually. The goal of the project is to provide a solution that will save art professionals time by offering a secure, encrypted database of artworks, a means to locate anything in the inventory and send it to clients at lightning speed.
